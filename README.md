# React Todo List - Improved Version

פרויקט רשימת מטלות ב-React הכולל את דרישות החובה וגם את דרישות הבונוס.

## מה יש בפרויקט

- הצגת רשימת מטלות קיימות
- הוספת מטלה חדשה דרך input וכפתור
- מניעת הוספה של מטלה ריקה
- מחיקת מטלה ספציפית
- סימון מטלה כבוצעה בעזרת checkbox
- הבדל ויזואלי למטלה שבוצעה באמצעות קו חוצה
- לכל מטלה יש id, text, isCompleted
- סינון לפי הכל / פעילות / בוצעו
- עריכת טקסט של מטלה קיימת
- שמירה ב-localStorage כך שהמידע נשמר גם אחרי רענון
- עיצוב רספונסיבי בסיסי

## איך מריצים

```bash
npm install
npm run dev
```

לאחר ההרצה נכנסים לכתובת שמופיעה בטרמינל, בדרך כלל:

```bash
http://localhost:5173
```

## קבצים מרכזיים

- `src/App.jsx` - כל הלוגיקה של ניהול המטלות והרכיבים.
- `src/App.css` - העיצוב של האפליקציה.
- `src/main.jsx` - חיבור React ל-HTML.
- `index.html` - קובץ HTML ראשי.
